<div align="LEFT"><span class="PCTabstract"><b class="notranslate">(EN)</b><span class="PCTabstract trans-section" lang="en">A waste collection system (10) includes one or more waste containers (20) for receiving waste (50), a server system (100) for receiving one or more signals via a wireless communication network (70) from sensor arrangements (30, 40) included on the one or more waste containers (20) for sensing a waste status of the one or more containers (20). Beneficially, the server system (100) is operable from receiving the one or more signals to compute an optimal strategy for one or more waste collection vehicles (150) to collect waste from the one or more waste containers (20). The waste collection system (10) is capable of supporting a competitive bidding arrangement for implementing collection of waste (50) for the one or more waste containers (20).</span><br/><b class="notranslate">(FR)</b><span class="PCTabstract trans-section" lang="fr">L'invention concerne un système (10) de collecte de déchets qui comprend un ou plusieurs conteneurs (20) à déchets destinés à recevoir les déchets (50), un système serveur (100) afin de recevoir un ou plusieurs signaux par le biais d'un réseau (70) de communication sans fil depuis des agencements de capteur (30, 40) inclus dans le ou les conteneurs (20) pour déchets, afin de détecter l'état des déchets du ou des conteneurs (20). De manière avantageuse, le système serveur (100) peut fonctionner pour recevoir le ou les signaux, afin de calculer une stratégie optimale pour un ou plusieurs véhicules (150) de collecte de déchets, afin de collecter les déchets depuis le ou les conteneurs (20) pour déchets. Le système (10) de collecte de déchets est capable de supporter un agencement d'offres compétitives pour mettre en œuvre la collecte des déchets (50) pour le ou les conteneurs (20) pour déchets.</span><br/></span></div>
